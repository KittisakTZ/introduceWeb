ผลงานที่เคยทำ

โปรเจกต์ที่เคยพัฒนา

  • ระบบจองตั๋วคอนเสิร์ต (Ticket4U): ระบบจองตั๋วที่พัฒนาขึ้นโดยใช้ React สำหรับฝั่งผู้ใช้ และ Node.js สำหรับฝั่งเซิร์ฟเวอร์ รวมถึงการจัดการฐานข้อมูลด้วย PostgreSQL มีฟีเจอร์สำหรับการค้นหาคอนเสิร์ต การจองตั๋ว และการจัดการการชำระเงิน ทำให้ผู้ใช้สามารถจองตั๋วได้อย่างรวดเร็ว
  • แอปพลิเคชันจัดการงานในทีม (TeamManager): แอปที่ออกแบบมาเพื่อช่วยให้ทีมสามารถจัดการงานและแบ่งหน้าที่กันได้ง่ายยิ่งขึ้น พัฒนาด้วย React และ Firebase มีฟีเจอร์การสร้างงาน การติดตามสถานะ และการสนทนาภายในทีม เพื่อเพิ่มประสิทธิภาพการทำงานร่วมกัน
  • เว็บไซต์ร้านค้าออนไลน์ (ShopEasy): ระบบร้านค้าออนไลน์ที่ใช้เทคโนโลยี (Express, React, Node.js) รองรับการเพิ่มสินค้าลงในตะกร้า การชำระเงินออนไลน์ และระบบสมาชิกที่ช่วยให้ร้านค้าจัดการสินค้าและผู้ใช้ได้ง่ายขึ้น
